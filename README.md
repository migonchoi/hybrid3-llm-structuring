# Hybrid3 LLM Extraction Prototype

This project explores how large language models can convert semi-structured
materials database pages into structured JSON suitable for machine learning.

## What it does

- Fetches a Hybrid3 material page
- Filters relevant HTML sections
- Uses LLM to extract structured information
- Outputs a JSON file

## Extracted fields

- product information
- dimensionality
- atomic coordinates
- synthesis method

## Motivation

Scientific databases are often human-readable but not machine-readable.
This project explores how LLMs can help structure messy scientific data.
